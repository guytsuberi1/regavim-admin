# event-flyer — פלייר אירוע ב-AI ("Nano Banana")

ה-Edge Function הזה מקבל נתוני אירוע, בונה prompt עשיר בעברית, ושולח ל-**Google Gemini
2.5 Flash Image ("Nano Banana")** — שמחזיר **תמונת פלייר מעוצבת** של האירוע/הטיול.

## פריסה (חד-פעמי)

המפתח כבר קיים (`GEMINI_KEY_ADMIN`, אותו אחד של `meeting-to-events`), אז צריך רק לפרוס:

```bash
supabase functions deploy event-flyer
```

אופציונלי — לשנות דגם בלי לגעת בקוד:
```bash
supabase secrets set GEMINI_IMAGE_MODEL=gemini-2.5-flash-image
```

## איך זה עובד
- הפרונט (כפתור **🎨 פלייר AI** בכרטיס האירוע) שולח את נתוני האירוע (כותרת, סוג, תאריך,
  שעות, יעד, קבוצה, לו"ז, הערות).
- הפונקציה בונה prompt מפורט בעברית בסגנון המותג (ירוק-זהב-קרם, לוגו רגבים, לו"ז, ברכה),
  שולחת ל-Nano Banana, ומחזירה תמונה (data URL).
- הפרונט מציג את התמונה עם הורדה / שיתוף / הדפסה.

## הערות
- **עלות/מכסה:** דגם התמונות עשוי לצרוך מכסה גדולה יותר מטקסט; פלייר בודד נמצא בדרך כלל
  בתוך המכסה החינמית, אך ייתכנו מגבלות יומיות.
- **בטיחות:** אם Gemini חוסם תוכן מסוים תוחזר הודעת שגיאה עם הסיבה.
- **אבטחה:** הפונקציה מאמתת JWT אוטומטית (רק משתמש מחובר יכול להריץ).
